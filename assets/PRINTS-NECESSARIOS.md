# Prints Necessários para o Portfólio

Coloque cada arquivo na pasta indicada com exatamente o nome da coluna **Arquivo**.
Formatos aceitos: `.jpg`, `.png`, `.webp` (ajuste a extensão no `index.html` se usar .webp).

## Como adicionar as imagens

1. Tire o print ou salve a imagem
2. Renomeie conforme a coluna **Arquivo** acima
3. Coloque na pasta correta (coluna **Pasta**)
4. Abra o site — as imagens aparecem automaticamente

> Se alguma imagem não existir, o espaço some automaticamente (sem imagem quebrada).
